# 数独

## 坐标系约定
   行、列编号从 `0 ~ n`。例如: 第一行是 `r= 0`, 最后一列是 `c = n`

## 公式
对任务给定格子 (r, c), 以 3 x 3 为例:

1. 行起点(row_start)
  ```
              | r |
  row_start = | - | x 3
              | 3 |
  ```
   理解: 
   - 把行号按每 3 行一组分组: 第 0 组: 0 .. 2, 第 1 组: 3 .. 5, 第 2 组: 6 .. 8
   - 取该组的第 1 个行号: `组号(r / 3) x 3`

2. 列起点(col_start)
   ```
               | c |
   row_start = | - | x 3
               | 3 |
   ```
   理解:
   - 把列号按每 3 行一组分组: 第 0 组: 0 .. 2, 第 1 组: 3 .. 5, 第 2 组: 6 .. 8
   - 取该组的第 1 个列号: `组号(c / 3) x 3`   
   
## 遍历宫内偏移
   1. 两重偏移
      - row_offset ∈ {0, 1, 2}
      - col_offset ∈ {0, 1, 2}
      - 宫内格子坐标: (row_start + row_offset, col_start + col_offset)
     
   2. 用单索引 k (0..8)的一维映射
      - k ∈ {0, 1, ..., 8}
      - row_offset = floor(k / 3)
      - col_offset = k % 3
      - 宫内格子坐标: (row_start + k / 3, col_start + k % 3)
      ps: k = 0 → (0, 0), k = 1 → (0, 1), ..., k = 8 → (2, 2)
